# Changelog

## 0.1.0 — Initial Release

- Sidebar worktree explorer with branch names and dirty/clean status
- Create worktree from new or existing branch
- Remove worktree with branch cleanup and unmerged change warnings
- Quick switch between worktrees via command palette
- Status bar showing current branch and worktree count
- Color theming per worktree window (title bar, activity bar, status bar)
- Auto-assign colors from a configurable palette, with manual override
- Copy files (e.g. `.env`, `node_modules`) into new worktrees
- Run post-create commands after worktree creation
- Filesystem watcher for automatic sidebar refresh
